# NoMansSky.ModTemplate
An Adaptation for the MBIN based BNS (Bomber's Natural Settlements) mod for No Man's Sky using [Reloaded 2](https://github.com/Reloaded-Project/Reloaded-II/releases/latest) and the [No Man's Sky API](https://github.com/gurrenm3/NoMansSky.Api)!

For more info on modding No Mans Sky, check out the modding wiki here: [https://github.com/gurrenm3/NoMansSky.Api/wiki](https://github.com/gurrenm3/NoMansSky.Api/wiki)
